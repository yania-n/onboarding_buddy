# **About This Roadmap**

This roadmap defines the recommended certification path for each IT role over a 3-year horizon. All exam fees are reimbursed by Nexora on first pass via the Learning Stipend in Workday. Study time is supported — discuss with your manager how to protect time for exam preparation.

|  | **CERTIFICATION ≠ PROMOTION** Certifications are learning milestones, not promotion requirements. They signal breadth and commitment to craft. Promotion is based on demonstrated performance, not certificates. |

# **Certification Roadmap by Role**

| **Role** | **Year 1 (Foundation)** | **Year 2 (Specialisation)** | **Year 3 (Advanced / Leadership)** |
| --- | --- | --- | --- |
| **Technical Product Owner (TPO)** | Professional Scrum Product Owner I (PSPO I) | Snowflake SnowPro Core; IIBA CBAP (optional) | Professional Scrum Product Owner II (PSPO II) |
| **Site Reliability Engineer (SRE)** | AWS Certified Solutions Architect – Associate | AWS Certified DevOps Engineer – Professional | Certified Kubernetes Administrator (CKA); Google SRE certification |
| **FinOps Analyst** | FinOps Certified Practitioner (FOCP); AWS Cloud Practitioner | AWS Certified Cloud Practitioner → SAA-Associate | FinOps Certified Professional (FCOP); Azure Cost Management Specialty |
| **Cloud Security Engineer** | AWS Certified Security – Specialty | Certified Cloud Security Professional (CCSP) | CISSP (after 5 years experience); ISO 27001 Lead Implementer |
| **Platform Engineer (IaC)** | HashiCorp Certified: Terraform Associate | AWS Certified Developer – Associate | Certified Kubernetes Application Developer (CKAD); AWS Solutions Architect – Professional |
| **Developer Experience (CI/CD)** | GitHub Actions Certification; HashiCorp Terraform Associate | AWS Certified Developer – Associate | Certified Kubernetes Application Developer (CKAD); Platform Engineering Foundation (PEF) |
| **Architect (TCP)** | AWS Certified Solutions Architect – Professional | AWS Certified Advanced Networking OR Database Specialty | TOGAF 10 (Enterprise Architecture); Google Professional Cloud Architect |

# **How to Claim Certification Reimbursement**

- Pass your exam — request your digital certificate or transcript from the certification body

- In Workday, go to My Learning > External Development > Add Completed Learning

- Upload the certificate, enter the exam cost, and submit

- Your manager approves the reimbursement request in Workday within 5 business days

- Reimbursement is added to your next payroll
