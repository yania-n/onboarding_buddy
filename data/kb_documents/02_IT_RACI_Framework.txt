# **How to Read This RACI**

R = Responsible (does the work), A = Accountable (one person; approves and owns the outcome), C = Consulted (provides input before decision), I = Informed (notified after decision).

|  | **ONE ACCOUNTABLE PERSON** Every activity has exactly one Accountable person. If two people are listed as Accountable, that is an error — escalate to the IT Director to clarify. |
